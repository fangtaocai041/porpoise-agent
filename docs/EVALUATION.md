## 优势

| # | 优势 | 说明 |
|---|------|------|
| 1 | 理论深度行业罕见 | 同时落地 BDI + MDP + ReAct + ToT/GoT + Reflexion + MAS Topology，每层有清晰代码映射 |

| 6 | 安全工程完整 | AST 检查 + SQL 注入防护 + 沙箱子进程隔离 + 人工审批 + JSONL 审计 |
| 7 | 测试体系成型 | 56 单元 + 79 工作流 + CI = 135 测试 100% |
| 8 | 代码干净 | 旧模块桩代码已清理 (16→6) |

## 不足

| # | 不足 | 影响 | 改进方向 |
|---|------|:--:|------|
| 1 | 知网/万方/维普盲区 | 高 | Zotero 中文插件或知网研学导出 |
| 2 | ChromaDB 为可选依赖 | 中 | 加入 `[full]` extras |
| 3 | 声学分析未实数据验证 | 中 | 真实 PAM 数据端到端测试 |
| 4 | MAS 动态路由未生产验证 | 中 | 真实多步任务验证 Debate/Dynamic |
| 5 | LLM 依赖无轻量替代 | 中 | 小型分类模型替代部分调用 |
| 6 | 硬编码路径 (D:/Reasonix/...) | 低 | 统一环境变量 |
| 7 | 单会话无持久化 | 低 | SQLite/JSON checkpoint |
| 8 | pytest 标记未使用 | 低 | 测试文件加 `@pytest.mark` |
